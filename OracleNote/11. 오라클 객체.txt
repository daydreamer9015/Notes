1. 뷰(View)
    1) 개념
        - 실제 데이터를 가지고 있지 않은 '가상의 테이블'임
        - 그 실체는 데이터베이스에 이름과 함께 저장된 select 쿼리문임
    2) 사용하는 이유
        - 편리성
        - 보안성
            - 원본 테이블에 대한 접근 권한을 주지 않고, 뷰를 통해서만 제한된 데이터에 접근하도록 허용할 수 있음
    3) 뷰 생성, 조회, 수정, 삭제
        - 생성
            - create view 뷰이름 AS SELECT 쿼리문;
        - 조회
            - SELECT * FROM 뷰이름
        - 삭제
            - DROP VIEW 뷰이름

2. ROWNUM (의사 컬럼)
    1) 오라클이 자동으로 부여하는 행 번호 (1부터 시작)
    2) TOP-N 쿼리 작성 시 사용
    3) 페이징 처리 구현

3. 시퀀스 (SEQUENCE)
    1) PK 자동 번호 생성기
        - 기본 키와 같이 고유한 값을 생성하는데 사용됨
    2) 동시성 문제 해결
