1. SELECT
    1) 기본 형식
        - SELECT [열] FROM [테이블]
          WHERE [열] = [조건값]
    2) WHERE 문으로 조건에 맞는 데이터 조회하기
        - = 연산자로 특정 값 조회
        - AND, OR
        - 비교연산자
            - <>, != ...
        - 논리연산자
            - IN 연산자
                - 여러 개의 OR를 간단하게 표시할 수 있음
                - 문자는 작은따옴표, 숫자는 그냥 사용
                - NOT IN으로 제외 조건도 가능함

2. NULL 값
    1) NULL이란 해당 컬럼의 값이 없다는 의미 (0과는 다름)
        - 해당 컬럼이 NULL 값을 허용하는 경우
    2) NULL 값을 가지고 있는 컬럼을 검색하려면 IS NULL
    3) NULL이 아닌 값을 가지고 있는 컬럼을 검색하려면 IS NOT NULL
