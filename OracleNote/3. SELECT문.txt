1. SELECT
    1) 기본 형식
        - SELECT [열] FROM [테이블]
          WHERE [열] = [조건값]
    2) WHERE 문으로 조건에 맞는 데이터 조회하기
        - = 연산자로 특정 값 조회
        - AND, OR
        - 비교연산자
            - <>, != ...
        - 논리연산자
            - IN 연산자
                - 여러 개의 OR를 간단하게 표시할 수 있음
                - 문자는 작은따옴표, 숫자는 그냥 사용
                - NOT IN으로 제외 조건도 가능함
            - BETWEEN
                - 범위에 있으면 데이터를 조회
                - 두 경계값을 포함하여 검색을 수행함

2. NULL 값
    1) NULL이란 해당 컬럼의 값이 없다는 의미 (0과는 다름)
        - 해당 컬럼이 NULL 값을 허용하는 경우
    2) NULL 값을 가지고 있는 컬럼을 검색하려면 IS NULL
    3) NULL이 아닌 값을 가지고 있는 컬럼을 검색하려면 IS NOT NULL

3. LIKE
    1) 일부 키워드만으로 조회하는 경우가 많음
    2) LIKE는 와일드카드로 지정한 패턴과 일치하는 문자열, 날짜, 시간 등을 조회할 때 사용함
    3) %
        - 특정 문자열을 포함하는 데이터 조회
        - 0 ~ n글자
            - 예) A% : A로 시작하는 모든 문자열
                  %A : A로 끝나는 모든 문자열
                  %A% : A가 포함된 모든 문자열
    4) _ : 1글자
       __ : 2글자

4. 집합 연산자
    1) UNION
        - 합집합 (중복 제거)
    2) UNION ALL
        - 합집합 (중복 포함)
    3) MINUS
        - 차집합 (첫번째 결과에서 두번째 결과를 제외함)
    4) INTERSECT
        - 교집합 (두 결과의 공통 부분)
