1. JSP (JavaServer Pages)
    1) HTML 안에 Java 코드를 추가할 수 있는 기술
    2) 서버에서 동적으로 HTML을 생성
    3) .jsp 파일 => 서블릿(.java 파일) => 컴파일(.class) => 실행(JVM)
    4) View 역할에만 사용하는 쪽으로 발전됨 (비즈니스 로직 X)

2. JSP 주요 구성 요소
    1) <% Java 코드 %>     <--- 스크립틀릿
        - Java 코드 실행 영역
    2) <%@ page %>
        - 페이지 설정 (인코딩, import 등)
    3) JSP 내장 객체 (자동 제공)
        - request, response, session, application, out, pageContext 등
    4) <%= 변수/메서드 %>    <--- 표현식 (Expression) : = 마크 들어가는 것 주의!!
        - 값을 html에 출력
