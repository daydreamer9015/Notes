* View 계층 개요

1. View 계층 역할
    1) UI 제공
    2) AJAX로 REST API 호출
    3) 동적으로 댓글 목록 렌더링
    4) 사용자 상호작용 처리

2. 사용 기술
    1) Thymeleaf : 서버 사이드 템플릿 엔진
    2) JavaScript (ES6+)
        - REST API 호출 : Fetch API 사용
        - 동적 DOM 조작 : 댓글 목록 렌더링
        - 사용자 이벤트 처리 : 작성, 수정, 삭제
        - 보안 : XSS 방지, CSRF 토큰 처리
    3) Fetch API : AJAX 통신
    4) CSS : 스타일링
    5) Font Awesome 아이콘

3. 댓글 UI 구성
    1) 댓글 섹션 헤더
        - 댓글 3개
    2) 댓글 작성 폼 (로그인 사용자)
        - 댓글 작성 버튼
    3) 댓글 내용
        - 작성자 작성날짜
        - 댓글 내용

4. JavaScript 전체 흐름
    1) 페이지 로드 (DOMContentLoaded)
    2) loadComments() - 댓글 목록 조회
    3) renderComments() - 화면에 댓글 표시
    4) 사용자 액션에 따라
        - 작성 버튼 클릭 - createComment()
        - 수정 버튼 클릭 - updateComment() -> saveComment()
