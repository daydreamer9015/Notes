* 댓글 기능 - Repository 계층

1. Repository 역할
    - 데이터베이스 액세스 계층
    - Entity에 대한 CRUD 작업 수행
    - 복잡한 쿼리를 메서드로 추상화

2. Spring Data JPA
    - JpaRepository를 상속받기만 하면 기본 CRUD 메서드를 자동으로 제공함
    - 자동 제공되는 메서드
        - save()        // 저장
        - findById()    // ID로 조회
        - findAll()     // 전체 조회
        - deleteById()  // ID로 삭제
        - count()       // 개수 조회
