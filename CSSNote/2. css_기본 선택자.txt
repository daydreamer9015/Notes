1. 기본 선택자
    1) 태그뿐만 아니라 웹 문서의 어떤 요소에도 적용함
    2) 선택자
        -    웹 문서에서 어느 부분에 스타일을 적용할 지 알려주는 것

2. 전체 선택자 (Universal Selector)
    1) 전체 요소에 스타일을 적용함
    2) 스타일을 문서의 모든 요소에 적용할 때 사용함
    3) 주로 모든 하위 요소에 스타일을 적용할 때 사용함
    4) 형식
        - * (별표) 사용함
        - * { 속성: 값; ....}
    5) 웹 브라우저의 기본 스타일을 초기화할 때 사용함
        - 패딩, 마진 여백을 0으로 지정할 수 있음.

3. 타입 선택자 (Type Selector)
    1) 특정 요소에 스타일을 적용하는 선택자
    2) 특정 태그를 사용한 모든 요소에 스타일을 적용함
        - 타입 선택자를 사용해서 스타일 지정하면 해당 태그를 사용한 모든 요소에 적용
    3) 형식
        -   태그명 { 스타일 규칙 }
    4) = 태그 선택자 (Tag Selector)
        -   태그 이름을 선택자로 사용함

4. 클래스 선택자 (Class Selector)
    1) 특정 부분에 스타일을 적용함
    2) 같은 태그라도 일부는 다른 스타일을 사용할 수 있음
    3) 클래스 이름을 사용해서 다른 선택자와 구별함
    4) 형식
        - 클래스 이름 앞에 마침표(.)를 반드시 붙여야 함
        - .클래스명 { 스타일 규칙 }
    5) 클래스 선택자가 문서에서 여러 번 적용할 수 있음

5. ID 선택자 (ID Selector)
    1) 특정 부분에 스타일을 한 번만 적용함 (특정 태그에 특정 스타일 적용)
    2) # 기호를 사용하여 스타일을 정의함
    3) 형식
        - #아이디명 { 스타일 규칙 }
    4) ID선택자는 문서에서 한번만 적용할 수 있음 

6. 같은 스타일 규칙을 사용하는 요소를 묶어서 표현
    1) 여러 선택자에 같은 스타일 규칙을 사용하는 경우
    2) comma(,)로 구분해 여러 선택자를 나열
    3) 형식
        - 선택자1, 선택자2, 선택자3 { 스타일 규칙 }
