1. 배경 이미지 지정
    1) 웹 요소에 배경 이미지를 넣는 속성
        - background-image : url('이미지 파일 경로')
    2) 배경 이미지의 반복 방법 지정하는 속성
        - backgroung-repeat
            -  속성값
                - repeat : 기본값. 브라우저 화면에 가득 들어찰 때까지 가로와 세로로 반복함
                - repeat-x : 브라우저 화면에 가득 들어찰 때까지 가로로 반복함 
                - repeat-y : 브라우저 화면에 가득 들어찰 때까지 세로로 반복함
                - no-repeat : 반복하지 않고 한번만 표시
    3) 배경 이미지의 위치를 조절하는 속성
        - background-position
            - 배경 이미지의 수평 위치 또는 수직 위치의 값을 지정함
            - 키워드 : left, center, right, top, bottom, center 등
            - 백분율 (%) : 요소의 왼쪽 모서리로부터 가로 %, 세로 % 위치
            - 크기 (px)
    4) 배경 이미지 크기를 조정하는 속성
        - background-size
            - 속성값
                - auto : 원래 배경 이미지 크기만큼 표시. 기본값
                - cover : 배경 이미지로 요소를 모두 덮도록 확대, 축소됨
                - <크기> : 이미지의 너비와 높이 지정함
                - <백분율>

2. gradation 효과로 배경 꾸미기
    1) 선형 그라데이션
        - 색상이 수직, 수평 혹은 대각선 방향으로 일정하게 변하는 것
        - linear-gradient(to <방향> 또는 <각도>, <색상 중지점>, [<색상 중지점>,......])
            - 방향 : left, right, top, bottom
        - 색상 중지점
            - parameter 하나가 더 들어감
            - linear-gradient(to <방향> 또는 <각도>, <시작 색상>, <색상 중지점의 색상이나 위치>, <끝 색상>)
    2) 원형 그라데이션
        - 원 또는 타원의 중심에서부터 동심원을 그리며 바깥 방향으로 색상이 변함 (기본값은 타원. 모양에 circle 혹은 ellipse를 넣어야 원형이 나옴)
        - 원의 중심과 크기를 지정하고 그라데이션 모양을 선택해야 함
        - radial-gradient(<모양><크기>, <색상 중지점>, [<색상 중지점>,......])
