1. 연결 선택자 [하위 선택자 / 자식 선택자 / 인접 형제 선택자 / 형제 선택자]
    0-1) 하위 요소
            - 특정 요소를 기준으로 그 안에 포함된 요소
    0-2) 자식 요소
            - 현재 요소를 기준으로 바로 한 단계 아래 요소
         손자 요소
            - 자식 요소의 한 단계 아래 요소
    0-3) 형제 요소
            - 부모 요소가 같은 경우
            - 먼저 나오는 요소를 형 요소, 나중에 나오는 요소를 동생 요소

    1) 하위 선택자 (Descendant selector)
        - 부모 요소에 포함된 하위 요소를 모두 선택
        - 자식 요소뿐 아니라 손자 요소, 손자의 손자 요소 등 모든 하위 요소까지 적용됨
        - 상위 요소와 하위 요소를 공백으로 구분함
        - 기본형
            - 상위요소 하위요소
            - 예) section p {}

    2) 자식 선택자 (child selector)
        - 하위 선택자와 다르게 자식 요소에만 스타일을 적용함
        - 기본형
            - 부모요소 > 자식요소

    3) 인접 형제 선택자 (adjacent selector)
        - 형제 요소 중에서 첫번째 동생 요소만 선택하는 것
        - 기본형
            - 요소1 + 요소2
            - 예) h1 + p { color: blue; }
                ---------
                h1의 형제인 p 요소 중에 첫번째 p

    4) 형제 선택자 (sibling selector)
        - 모든 형제 요소에 적용됨
        - 기본형
            - 요소1 ~ 요소2
            - 예) h1 ~ p { color: blue; }
                 --------
                h1의 형제인 모든 p 요소

2. 속성 선택자
    1) 특정 속성이 있는 요소를 선택하는 [속성] 선택자
        - 대괄호( [] ) 사이에 원하는 속성을 입력하면 됨
        - 예)
            - a[href] { 원하는 스타일 }
    
    2) 특정 속성값이 있는 요소를 선택하는 [속성=속성값] 선택자
        - 예)
            - a[target = _blank]
            - a 요소 중에서 target 속성값이 _blank인 것만 선택함

    3) 여러 값 중에서 특정 속성값이 포함된 속성요소를 선택 [속성~=값] 선택자

    4) 특정 속성값이 포함된 요소를 선택하는 [속성|=값] 선택자
        - 속성값은 한 단어로 일치해야 함
    
    5) 특정 속성값으로 시작하는 속성 요소를 선택하는 [속성^=값]
        - 속성값이 정확하게 일치하지 않더라도 지정한 속성값으로 시작하는 요소를 찾는 선택자

    6) 특정한 값으로 끝나는 속성의 요소를 선택하는 [속성$=값] 선택자
    
    7) 일부 속성값이 일치하는 요소를 선택하는 [속성*=값] 선택자
        - 속성값이 어느 위치에 있든지 지정한 속성값이 포함되어 있다면 해당 요소를 선택함

3. 가상(클래스) 선택자 (Pseudo element)
    1) 사용자가 웹 요소를 클릭하거나 마우스 포인터를 올려놓는 등 특정 동작을 할 때 적용되는 스타일이 바뀌도록 만들고 싶을 때 사용하는 선택자
        - :link
            - 아직 방문하지 않은 링크에 스타일 적용함
        - :visited
            - 방문한 링크에 스타일 적용함
        - :hover
            - 특정 요소에 마우스 포인터를 올려놓게 되면 스타일을 적용함
        - :active
            - 웹 요소를 활성화했을 때 스타일을 적용함
            - 어떤 웹 요소의 링크를 클릭하는 그 순간의 스타일 
        - :focus
            - 특정 웹 요소에 초점이 맞추어졌을 때 스타일을 적용함

    2) :checked
        - 선택한 항목의 스타일을 적용함
            - 폼의 라디오 버튼이 체크박스 버튼에서 선택한 항목에 checked라는 속성이 추가됨
            - 이렇게 checked 속성이 있는 요소의 스타일 지정함
    
    3) 구조 가상 클래스의 선택자
        - :first-child      여러 요소 중 첫번째 자식 요소를 선택함
          :last-child       여러 요소 중 마지막 자식 요소를 선택함
          :only-child       여러 요소 중에서 자식 요소가 하나일 때 선택함
          :nth-child(숫자 or 수식)      지정한 위치의 자식 요소를 선택함
         A:nth-child(숫자 or 수식)      A요소를 기준으로 지정한 위치의 자식 요소를 선택함

4. 가상 요소
    1) 문서의 특정 부분에 가상 요소를 만들어서 추가함
    2) 실제 내용에는 영향을 주지 않으면서 눈에 띄게 꾸며줄 때 사용 (가시적인 효과는 있음)
    3) 태그를 사용하지 않고 필요할 때마다 만들어서 사용함
    4) 가상 클래스와 구별하도록 이름 앞에 콜론을 두 개(::) 붙여서 표시함
    5) ::first-line
        - 첫번째 줄에 스타일을 적용
       ::first-letter
        - 첫번째 글자에 스타일을 적용
    6) ::before
        - 내용 앞에 컨텐츠를 추가하는 요소
       ::after
        - 내용 뒤에 컨텐츠를 추가하는 요소

        - 둘 다 content 속성을 사용해야 함
