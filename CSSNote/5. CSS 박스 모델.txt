0. 블록 레벨 요소와 인라인 레벨 요소
    1) Block Level
        - 태그를 사용해 요소를 html 문서에 삽입했을 때 혼자 한 줄을 차지하는 것을 가리킴
        - 해당 요소의 너비가 100%라는 뜻
            - 블록 레벨 요소의 좌우에 다른 요소가 올 수 없음
        - <h1>, <div>, <p>
    2)inline level
        - 한 줄을 차지하지 않음
        - 영역을 컨텐츠만큼만 차지함
            - 나머지 공간에는 다른 요소가 올 수 있음
        - 예) <span>

1. 박스 모델
    1) 모든 HTML 요소는 박스(상자) 형태로 화면에 표시됨
    2) 4가지 구성요소
        - Contents(내용) : 실제 텍스트나 이미지가 들어가는 영역
            - width와 height 속성으로 크기 조절
            - 기본적으로 내용에 맞춰 크기가 결정됨
        - Padding(패딩) : 내용과 테두리 사이의 내부 여백
            - 클릭 가능한 영역에 포함됨
        - Border(테두리) : 요소를 둘러싸는 경계선
            - 두께, 스타일, 색상을 지정할 수 있음
            - 요소의 전체 크기에 포함됨
        - Margin(마진) : 다른 요소와의 외부 여백   

2. 박스 모델 기본 구성
    1) 웹 문서의 블록 레벨 요소와 인라인 레벨 요소는 모두 박스 형태임
    2) 스타일시트는 이렇게 박스 형태인 요소를 박스 모델(box model) 요소라고 함
    3) 박스 모델은 컨텐츠 영역, 박스와 컨텐츠 영역 사이의 여백인 패딩(padding), 박스의 테두리(border), 그리고 여러 박스 모델 사이의 여백인 마진(margin) 등의 요소로 구성됨

3. Contents 영역의 크기 지정하는 속성
    1) width, height
        - 속성값
            - 너비나 높이 값을 px이나 em단위로 지정함
        - 백분율
            - 박스 모델을 포함하는 부모 요소를 기준으로 너비값 또는 높이값을 백분율(%)로 지정함
        - auto
            - 박스 모델의 너비값, 높이값이 컨텐츠 양에 따라 자동으로 결정됨. 기본값
    2) 박스 모델의 크기를 계산하는 속성
        - box-sizing
            - border-box : 패딩과 테두리를 포함해서 너비값을 지정함
            - content-box : 컨텐츠 영역만 너비값으로 지정함. 기본값
        - width, height 속성은 박스 모델에서 컨텐츠 주변의 여백(margin)이나 테두리를 뺀 나머지 컨텐츠 영역의 크기를 가리킴
        - 웹 문서에 여러가지 요소를 배치할 때 실제 박스 모델이 차지하는 크기는 컨텐츠 영역 외에도 컨텐츠와 테두리 사이의 여백(padding), 테두리 두께까지 계산해야 함

4. 박스 모델에 그림자 효과 주는 속성
    1) box-shadow :수평 거리, 수직 거리, 흐림 정도, 번짐 정도, 색상
    2) 사진 주변에 그림자 효과를 낼 수 있음

5. 박스 모델의 방향
    1) 상하좌우 4개 방향이 있어서 테두리, 마진, 패딩 등을 지정할 때 한꺼번에 똑같이 지정할 수도 있고 아니면 다 다르게 지정할 수도 있음
    2) 속성값이 top->right->bottom->left처럼 기본적으로 시계방향으로 적용됨

6. 테두리 스타일 지정하는 속성
    1) border-style

7. 테두리 두께 지정하는 속성
    1) border-width

8. 테두리 색상 지정하는 속성
    1) border-color

9. 테두리 스타일을 묶어 지정하는 속성
    1) border

10. 테두리 꼭지점을 둥글게 만드는 속성
    1) border-radius
        - 속성값
            - 크기 : px, em 단위와 수치로 표시
            - 백분율 : 현재 요소의 크기를 기준으로 비율(%)로 지정함
    2) 꼭지점마다 따로 둥글게 처리하기
        - border-top(혹은 bottom)-left(혹은 right)-radius

11. 여백을 조절하는 속성
    1) 요소 주변의 여백을 설정하는 속성
        - margin
            - 속성값
                - 크기 : px, em
                - 백분율 : 박스 모델을 포함한 부모 요소를 기준으로 너비값이나 높이값을 퍼센트(%)로 지정함
                - auto : display 속성에서 지정한 값에 맞게끔 적절한 값을 자동으로 지정함
    2) margin 속성을 사용하여 웹문서 가운데 정렬
        - 웹문서 전체를 화면 중앙에 배치
        - 우선적으로 배치할 요소의 너비값이 정해져 있어야 함
        - margin-left, margin-right의 속성값을 auto로 지정함
    3) margin 중첩
        - 요소를 세로로 배치할 경우에 각 요소의 margin과 margin이 만나면 margin 값이 큰 쪽으로 겹쳐짐 (margin 상쇄)

12. 컨텐츠와 테두리 사이의 여백 설정하는 속성
    1) padding(패딩)
