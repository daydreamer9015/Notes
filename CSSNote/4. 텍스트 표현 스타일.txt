1. 글꼴 관련 스타일
    1) font-family
        - 글꼴 지정하는 속성
        - 시스템 글꼴을 이용해 웹 브라우저 화면에 표시
        - <body> 태그 스타일에서 일단 한번 정의
            - 문서 전체에 적용됨
            - 문서 안의 모든 자식 요소에 같은 글꼴 계속 사용함
    2) font-size
        - 글자 크기를 지정하는 속성
        - 글자 크기를 따로 지정하지 않더라도 제목 텍스트는 본문 텍스트보다 굵고 크게 표시됨
            - 웹 브라우저마다 기본 스타일이 적용되어 있음
        - 절대 크기로 글자 크기 지정하기 (px)
        - 상대 크기로 글자 크기 지정하기
            - 기준값이 있고 그 값의 몇 배인지를 표시하는 방법
            - %, em 
                - 부모 요소의 글씨 크기를 기준으로 상대적인 값 지정함
                - 예)
                    - 부모 요소 font-size: 20px
                    - 자식 요소 font-size: 15px
                        - 20 * 1.5 = 30px
    3) font-style
        - 이탤릭체로 글자를 표시하는 속성
    4) font-weight
        - 글자 굵기를 지정하는 속성

2. 웹 폰트와 아이콘 폰트
    1) 웹 폰트
        - 웹 문서에서 사용한 글꼴 정보를 서버에 올려놓고, 사용자가 사이트에 접속하면 웹문서를 내려받으면서 웹 폰트도 사용자 시스템으로 내려받는 형식임
    2) 아이콘 폰트
        - 아이콘을 이미지가 아닌 글자 형태로 사용
        - 웹에 올라와 있는 아이콘 폰트 정보를 링크해서 사용

3. 텍스트 관련 스타일
    1) color 속성
        - 글자색 지정
        - 16진수로 표현하는 방법
            - #ffff00 (0~9, a~f)
            - #000000(검은색) ~ #ffffff(흰색)
            -예) #0000ff 두 자리씩 중복될 경우 줄여서 표기 가능(#00f)
        - hsl, hsla로 표현하는 방법
            - 색상(hue), 채도(saturation), 명도(lightness), 투명도(alpha)
        - 색상을 영문 이름으로 표현하는 방법
    2) text-align 속성
        - 텍스트 정렬
            - 정렬 지정하지 않으면 (왼쪽 정렬)
            - 가운데 정렬 (center)
            - 양쪽 정렬 (justify)
    3) line-height 속성
        - 줄 높이를 조절하는 속성
        - 한 문단이 두 줄을 넘으면 사이에 간격이 생김
            - 줄 높이가 너무 좁거나 넓으면 가독성이 떨어짐
        - 텍스트를 세로 정렬할 때 유용함
            - line-height 속성값을 영역의 height 값과 똑같이 지정
    4) text-decoration 속성
        - 텍스트에 밑줄, 윗줄, 취소선을 표시하는 속성
    5) text-shadow 속성
        - 텍스트에 그림자 효과를 추가하는 속성
        - 사이트 제목처럼 강조해야 할 글자에 사용하면 입체감 있게 보여줌
    6) letter-spacing, word-spacing 속성
        - 글자와 글자 사이 간격(자간)
        - 단어와 단어 사이의 간격

4. 목록 스타일
    1) list-style-type
        - 불릿 모양과 번호 스타일 지정하는 속성
    2) list-style-image
        - 불릿 대신 이미지를 사용하는 속성
    3) list-style-position
        - 목록 들여쓰는 속성

5. 표 스타일
    1) caption-side
        - 표 제목의 위치 정해주는 속성
    2) border
        - 표 테두리 그려주는 속성
    3) border-collapse
        - 표와 셀 테두리를 합쳐주는 속성
