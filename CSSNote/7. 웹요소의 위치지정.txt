1. 웹 요소의 위치를 정하는 속성
    1) left
        - 기준 위치와 요소 사이에 왼쪽으로 떨어져 있는 정도를 숫자로 지정할 때 쓰는 속성 
    2) right
        - 기준 위치와 요소 사이에 오른쪽으로 떨어져 있는 정도를 숫자로 지정할 때 쓰는 속성
    3) top
        - 기준 위치와 요소 사이에 위쪽으로 떨어져 있는 정도를 숫자로 지정할 때 쓰는 속성
    4) bottom
        - 기준 위치와 요소 사이에 아래쪽으로 떨어져 있는 정도를 숫자로 지정할 때 쓰는 속성

2. 배치 방법을 지정하는 속성
    1) position
        - 요소를 자유자재로 배치해 줌
        - 속성값
            - static : 문서의 흐름에 맞추어 배치함 (기본값)
            - relative : 위치값을 지정할 수 있음. 그 외는 static과 같음
            - absolute : relative 값을 사용한 상위 요소를 기준으로 위치를 지정하여 배치함
            - fixed : 브라우저 창을 기준으로 위치를 지정해 배치함 (고정되어 움직이지 않음)
