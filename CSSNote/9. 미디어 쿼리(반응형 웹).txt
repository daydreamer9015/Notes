1. 반응형 웹 디자인 (Responsive Web Design)
    1) PC나 노트북, 스마트폰에서 웹 사이트에 접속하는 경우, 다양한 화면 크기에 맞게 표시
        - 웹 요소를 화면 크기에 맞게 재배치하고 표시 방법만 바꾸어 사이트를 구현해 줌
    2) 뷰포트 (Viewport)
        - 뷰포트를 지정하면 접속한 기기의 화면에 맞추어 확대하거나 축소해서 표시할 수 있음
        - <meta name="viewport" content="width=device-width, initial-scale=1.0">
        - 뷰포트 단위
            - vw (viewport width) : 1vw는 뷰포트 너비의 1%에 해당됨
              vh (viewport height) : 1vh는 뷰포트 높이의 1%에 해당됨
            - 웹 요소를 뷰포트에 가득 채우려면 
                => 너비를 100vw, 높이를 100vh로 설정하면 됨

2. 반응형 요소 만들기
    1) 상대값으로 글자 크기 지정하기
        - em 단위, rem 단위
    2) em 단위
        - 부모 요소에서 지정한 글꼴의 대문자 M의 너비를 1em으로 놓고 이에 비례하여 상대적으로 값을 계산함
        - 부모 요소에서 글자 크기를 지정하지 않으면 body 요소의 기본 크기를 사용함
            - 웹문서에서 body의 기본 글자 크기는 16px이고 이 값이 1em에 해당함
    3) rem 단위
        - root em
        - 루트에서 지정한 크기를 기준값으로 사용함

3. 반응형 이미지 만들기
    1) 부모 요소 너비에 따라 너비와 높이가 바뀌는 이미지
    2) 이미지는 웹 문서에 삽입할 때부터 크기가 정해져 있으므로 브라우저 화면이 작아지면 이미지의 일부만 보임
    3) width: 100% 사용
        - 이렇게 지정하면 이미지가 부모 요소의 너비에 맞춰 항상 100%가 됨
        - 부모 요소의 너비가 너무 크면 원본보다 커져서 화질이 떨어질 수 있음
    4) max-width: 100% 사용
        - 부모 요소의 크기에 따라 이미지가 커질 때 원본 이미지만큼 커짐

4. 미디어 쿼리
    1) Media Queries
        - 사이트에 접속하는 장치에 따라 특정한 CSS 스타일을 사용하는 방법 
        - 미디어 쿼리를 사용하면 접속하는 기기의 화면 크기에 따라 레이아웃이 달라짐
    2) 미디어 쿼리 구문
        - @media 속성을 사용하여 특정 미디어에서 어떤 CSS를 적용할 것인지를 지정해 줌
        - @media [only / not] 미디어 유형 [and 조건] * [and 조건]
                  ----   ---             ---
            - only : 미디어 쿼리를 지원하지 않는 웹 브라우저에서는 미디어 쿼리를 무시하고 실행하지 않음
            - not : not 다음에 지정하는 미디어 유형을 제외함
            - and : 조건을 여러 개 연결해서 추가할 수 있음
        - 예)
            @media screen and (min-width: 768px) and (max-width: 1439px) {
                /적용할 CSS 스타일/
            }
    3) 미디어 유형
        - 미디어 별 적용할 CSS를 따로 작성함
        - @media 속성의 미디어 유형 종류
            - all : 모든 미디어 유형에서 사용할 CSS를 정의 / 기본값
            - screen : 컴퓨터 스크린에서 사용할 CSS를 정의 / 스마트폰
            - print
            - tv
            - tty
            - handheld
            - projection
        -웹 문서의 가로 너비와 세로 높이 속성
            - 실제 웹 문서 내용이 화면에 나타나는 영역(뷰포트)
            - 뷰포트의 가로 너비와 세로 높이를 미디어의 쿼리 조건으로 사용할 수 있음
            - 속성
                - width, height
                - min-width, min-height (최소 가로 너비, 최소 세로 높이 지정)
                - max-width, max-height (최대 가로 너비, 최대 세로 높이 지정)

                - 화면 회전 속성
                    - orientation: portrait (세로 모드)
                      orientation: landscape (가로 모드)

    4) 미디어 쿼리 중단점(Breakpoint)
        - 미디어 쿼리 작성할 때 화면 크기에 따라 서로 다른 CSS를 적용할 분기점
        - 대부분 기기의 화면 크기를 기준으로 함
        - 모바일, 태블릿, 데스크탑 정도로만 구분함
            - 처리 속도, 화면 크기 등에서 모바일 제약 조건이 더 많음
                - 레이아웃을 기본으로 하여 CSS를 만듦
            - 화면이 큰 태블릿, 데스크탑에 맞춰 더 많은 기능과 스타일 추가함
            - 스마트폰
                - min-width의 세로와 가로를 portrait 320px, landscape 480px로 지정함
            - 태블릿
                - min-width의 세로와 가로를 portrait 768px 이상, landscape 1024px 이상으로 지정함
            - 데스크탑
                - 화면 크기가 1024px 이상이면 데스크탑으로 설정

5. 미디어 쿼리 적용하기
    1) 외부 CSS 파일 연결하기
        - <link rel="stylesheet" href="css/style.css">
    2) 웹 문서에 직접 정의하기
        - <style media="screen and (max-width: 320px)">
            body {
                background-color: orange;
            }
          </style>
        
        - <style>
            @media screen and (max-width: 320px) {
                body {
                    background-color: orange;  
                }
            }
          </style>
