1. intro
    1) 웹 문서에 여러 요소가 있을 때 플렉스 박스 레이아웃을 사용하면 요소를 배치하기도 쉽고 반응형으로 동작하게 할 수 있음
    2) 플렉스 박스, 플렉스 컨테이너 개념 파악
    3) 배치할 때 사용하는 속성들 파악
    4) 반응형으로 만들 때 유용한 속성들 파악
    5) Flexbox
        - CSS의 레이아웃 방법 중 하나
        - 컨테이너 안의 아이템들을 효율적으로 배치하고 정렬할 수 있게 해주는 강력한 도구임
            - 수평/수직 정렬 쉬움
            - 반응형 레이아웃을 쉽게 만들 수 있음
            - 아이템의 크기를 자동으로 조절 가능함

2. Flexbox Layout에서 사용하는 용어
    1) 플렉스 컨테이너 Flex Container (부모 박스)
        - 플렉스 박스 레이아웃 적용할 대상을 묶는 요소
    2) 플렉스 항목 (자식 박스)
        - 플렉스 박스 레이아웃을 적용할 대상
        - 1 ~ 6까지 작은 박스들이 모두 해당됨
    3) 주축 (main axis)
        - 플렉스 컨테이너 안에서 플렉스 항목을 배치하는 기본 방향임
        - 기본적으로 왼쪽에서 오른쪽으로 수평 방향으로 배치함
        - 플렉스 항목의 배치가 시작되는 위치: 주축 시작점
          플렉스 항목의 배치가 끝나는 위치 : 주축 끝점
    4) 교차축 (cross axis)
        - 주축으로 배치하다가 끝점까지 닿았을 때 어떻게 배치할 지 결정하는 속성
        - 기본적으로 위에서 아래로 배치함
        - 플렉스 항목의 배치가 시작되는 위치 : 교차축 시작점
          플렉스 항목의 배치가 끝나는 위치 : 교차축 끝점

3. 플렉스 컨테이너에서 사용하는 속성
    1) 배치 속성
        - justify-content : 주축의 정렬 방법
        - align-items : 교차축의 정렬 방법
        - align-content : 교차축에 여러 줄로 표시할 때 사용하는 정렬 방법
        - align-self : 플렉스 항목을 각각 따로 정렬함
    2) 플렉스 컨테이너를 지정하는 속성
        - display
            - 속성값
                - flex : 플렉스 컨테이너를 블록 레벨로 만듦
                - inline-flex : 플렉스 컨테이너를 인라인 레벨로 만듦
    3) 플렉스 방향을 지정하는 속성
        - flex-direction
            - 속성값
                - row
                    - 기본값
                    - 주축을 가로로 지정하고 왼쪽에서 오른쪽으로 지정함 (1-2-3)
                - row-reverse
                    - 주축을 가로로 지정하고 오른쪽에서 왼쪽으로 지정함 (3-2-1)
                - column
                    - 주축을 세로로 지정하고 위에서 아래로 지정함
                - column-reverse
                    - 주축을 세로로 지정하고 아래에서 위로 지정함

4. 플렉스 항목의 줄을 바꾸는 속성 - flex-wrap
    1) 플렉스 컨테이너 너비보다 플렉스 항목(아이템)이 더 많은 경우에 줄바꿈 여부를 지정함
    2) 속성값
        - nowrap : 플렉스 항목을 한 줄로 표시함 (기본값)
        - wrap : 플렉스 항목을 여러 줄로 표시함
        - wrap-reverse : 플렉스 항목을 여러 줄로 표시하는데 교차축의 시작점과 끝점이 바뀌는 역순으로 표시됨

5. 배치 방향과 줄 바꿈을 한꺼번에 지정하는 속성 : flex-flow = flex-direction + flex-wrap

6. 주축 정렬 방법 지정하는 속성 : justify-content
    1) 주축에서 플렉스 항목 간의 정렬 방법 지정함
    2) 속성값
        - flex-start : 기본값. 주축의 시작점에 맞춰 정렬함
        - flex-end : 주축의 끝점에 맞춰 정렬함
        - center : 주축의 중앙에 맞춰 정렬함
        - space-around : 항목을 고르게 정렬함. 각 항목은 양쪽 여백이 생기는데 그 여백의 절반만큼 자리를 차지함
        - space-between : 첫 항목은 주축 시작점에, 마지막 항목은 주축 끝점에 배치한 후 나머지 항목은 같은 간격으로 정렬함
        - space-evenly : 항목을 고르게 정렬함. 각 항목의 여백은 모두 동일함

7. justify-content 속성과 margin 속성 함께 사용

8. 교차축 정렬 방법 지정 속성 : align-items
    1) 속성값
        - flex-start : 교차축의 시작점에 맞춰 정렬
        - flex-end : 교차축의 끝점에 맞춰 정렬
        - center : 교차축의 중앙에 맞춰 정렬
        - baseline : 교차축의 문자 기준선에 맞춰 정렬
        - stretch : 플렉스 항목을 늘려 교차축에 가득 차게 정렬 <= 기본값

9. 특정 항목만 정렬 방법 지정하는 속성 : align-self
    1) 속성값
        - auto : 기본값. 부모요소 align-items 값을 사용해서 정렬
        - flex-start : 플렉스 항목의 시작 위치에 맞춰 정렬. 텍스트 포함하고 있을 경우 언어의 방향에 따라 시작 위치가 결정됨
        - flex-end : 플렉스 항목의 끝 위치에 맞춰 정렬. 텍스트 포함하고 있을 경우 언어의 방향에 따라 끝 위치가 결정됨

10. 여러 줄 일 때 교차축 정렬 방법 지정하는 속성 : align-content
    1) 주축에서 줄 바꿈이 생겨서 플렉스 항목을 여러 줄로 표시할 때 사용함
