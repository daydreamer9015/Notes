1. 플랫폼으로서의 웹
    1) 디지털 트랜스폼 Digital Transform
        - 오프라인에 있는 모든 정보와 서비스를 온라인으로 옮기는 작업

2. 네트워크의 네트워크
    1) 라우터(router) : 네트워크와 네트워크를 효율적으로 연결함
    2) 서버와 클라이언트 (server, client) : 정보를 제공하는 서버, 정보를 제공받는 클라이언트

3. 프런트엔드 개발과 백엔드 개발
    1) 프런트엔드(frontend)
        - 사용자 앞에 보이는 영역
        - 웹 브라우저 화면에 보이는 것
        - HTML, CSS, JavaScript
    2) 백엔드(backend)
        - 사용자 뒤에 있어서 보이지 않는 영역
        - 서버에 데이터베이스 형태로 정보는 저장됨
        - 자바, 노드, js 

4. 웹 기본 동작
    1) IP 주소와 도메인, DNS 서버
                          도메인            IP 주소
        - 사용자(브라우저) <------> DNS 서버 <------> 서버
    2) HTTP(S) 프로토콜
        - HTTP
            - 텍스트 데이터 전송
        - HTTPS
            - 암호화된 데이터 전송

5. 웹 접근성
    1) 시각 장애인은 화면 낭독기로 웹을 이용할 수 있어야 함
    2) 나이와 장애에 상관없이 모든 사용자가 접근할 수 있어야 함
