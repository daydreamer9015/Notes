1. 텍스트 입력하기
    1) 제목을 나타내는 <hn> 태그
        - h는 제목(heading)의 줄임말
        - n의 자리에는 1~6의 숫자가 들어감
            - 제목 텍스트 크기별 표시
            - <h1>이 가장 큰 제목
    2) 텍스트 단락을 만드는 <p> 태그, 줄 바꾸는 <br> 태그
    3) 분위기를 전환하는 <hr> 태그
        - horizontal line의 줄임말
        - 가로줄 생성하는 용도
    4) 인용할 때 쓰는 <blockquote> 태그
        - 큰 따옴표("")를 사용해 표시
            - 다른 사람의 말이나 책의 내용 인용
    5) 텍스트를 강조하거나 굵게 표시하는 <strong>, <b> 태그
        -b=bold
    6) 기울기 텍스트를 표시하는 <em>, <i>, <cite>

2. 목록 만들기
    1) 순서 목록을 만드는 <ol>, <li> 태그 : ordered list
    2) 순서 없는 목록을 만드는 <ul>, <li> : unordered list

3. 표(table) 만들기
    1) 표의 구성 요소
        - 행(row), 열(column)이 만나는 셀(cell)로 이루어짐
    2) 표를 만드는 <table> 태그
    3) 행을 만드는 <tr> 태그와 셀을 만드는 <td>, <th>
    4) 표의 구조를 지정하는 <thead>, <tbody>, <tfoot> 태그 : <thead> 제목, <tbody> 본문, <tfoot> 요약
    5) 행이나 열을 합치는 rowspan, colspan 속성
        - <td rowspan="합칠 셀의 개수">내용</td> (행)
        - <td colspan="합칠 셀의 개수">내용</td> (열)
    6) 열을 묶어주는 <col>, <colgroup> 태그
        - <col> 태그는 열을 1개만 지정할 때 사용
        - <colgroup> 태그는 <col> 태그를 2개 이상 묶어서 사용함

4. 이미지 삽입
    1) 이미지를 삽입하는 <img> 태그
    2) <img src="이미지 파일 경로" alt="대체 텍스트">
    3) 이미지 크기를 조절하는 width, height 속성
        - 이미지 원래 크기대로 표시됨
        - 만약 이미지 파일은 수정하지 않고, 웹 브라우저 창에서 보이는 이미지 크기만 조절함
        - width는 이미지의 너비, height는 이미지의 높이를 지정
        - 속성은 2개 모두 사용할 수도 있고 1개만 사용할 수도 있음
            - 둘 중 하나만 지정하면 나머지 속성은 비율을 자동으로 계산하여 나타냄
        - 단위는 퍼센트(%)가 있고 픽셀(px)이 있음 (default는 px)
            - 숫자만 입력하면 단위는 px임
            - % : 현재 웹 브라우저 창의 너비와 높이를 기준으로 이미지 크기를 결정함
            - px : 해당 픽셀 크기만큼 표시함

5. 오디오와 비디오 파일 삽입하는 <audio> <video> 태그
    1) <audio src="오디오 파일 경로"></audio>
    2) <video src="비디오 파일 경로"></video>
