2. Array 객체로 배열 만들기
    1) 초기값이 없는 경우
        - 예)
            let arr1 = new Array();     // 배열 크기 미지정
            let arr2 = new Array(5);    // 배열 크기 지정함

    2) 초기값이 있는 경우
        - 예)
            let arr3 = ["one", "two", "three", "four", "five"];
            let arr4 = Array("one", "two", "three", "four", "five");

4. Array 객체의 method
    1) concat()
        - 기존 배열에 요소를 추가해 새로운 배열 만듦
        - 배열 합치기
    2) join()
        - 배열 안의 요소를 문자열로 만듦
        - 배열 요소를 연결해서 하나의 문자열로 만들어 줌
            - 각 요소 사이에 원하는 구분자('/', ',')를 넣을 수 있음
                - 따로 지정하지 않으면 요소를 쉼표(,)로 구분함
    3) push()
        - 새로운 요소를 추가함 (요소는 여러 개 일 수 있음)
        - 배열 맨 끝에 요소를 추가함
        - 원래 있던 배열이 바뀜
    4) unshift()
        - 새로운 요소를 추가함(요소는 여러 개 일 수 있음)
        - 배열 맨 앞에 요소를 추가함
        - 원래 있던 배열이 바뀜
    5) pop()
        - 배열에서 뒤에 있는 요소를 꺼냄
        - 기존 배열은 꺼낸 요소가 빠진 상태로 변경됨
    6) shift()
        - 배열에서 앞에 있는 요소를 꺼냄
        - 기존 배열은 꺼낸 요소가 빠진 상태로 변경됨
