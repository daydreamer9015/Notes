2. 이벤트 처리기  
    1) 웹 문서에서 이벤트가 발생하면 처리하는 함수를 이벤트 처리기 또는 이벤트 핸들러라고 함  
    2) HTML 태그 안에서 on 다음에 이벤트명을 붙여서 속성 이름을 만듦 
        - 그 다음에 실행할 이벤트 처리기의 함수명을 작성하면 됨 
    3) 기본형 
        - <태그 on이벤트명 = "함수명">

3. addEventListener()
    1) HTML 요소에 이벤트(클릭, 입력, 키보드 등)가 발생했을 때 실행할 코드를 등록하는 함수임
    2) 기본 문법
        - 요소. addEventListener("이벤트종류", 실행할 함수);
        - 예) 이벤트종류: 'click', 'keydown', 'mouseover'...
              실행할 함수 : 이벤트가 발생했을 때 실행할 코드(함수)

4. try~catch
    1) JavaScript에서 에러(예외)를 처리하는 핵심 문법임
    2) 프로그램이 실행 중 에러가 발생해도 중단되지 않고 계속 실행될 수 있도록 해줌
    3) 기본 문법
        - try {
            // 에러가 발생할 수 있는 코드
        } catch(error) {
            // try구문에서 에러가 발생했을 때 실행할 코드
        } finally {
            // 에러 발생 여부에 관계없이 반드시 실행되는 코드 (선택사항임)
        }
