1. JavaScript로 무엇을 할 수 있나?
    1) 웹의 요소를 제어함
        - 예) 메인 메뉴 위로 마우스 포인터를 올리면 숨어있던 하위 메뉴가 펼쳐짐
        - 예) 회원 가입, 메뉴, 장바구니
            - 입력 정보가 형식에 맞는지 체크
    2) 웹 애플리케이션 만들기
    3) 다양한 라이브러리를 사용할 수 있음 (웹 개발을 도와줌)
        - 리액트, 뷰, 제이쿼리 등
    4) 서버 개발을 할 수 있음

2. 웹 브라우저 + JavaScript의 조합
    1) 웹 문서 안엔 <script> 태그로 자바스크립트 작성할 수 있음
        - 웹 문서 안의 어디든 위치할 수 있음
        - 삽입된 위치 그 자리에서 바로 스크립트가 실행됨
        - <script> 태그는 하나의 문서에서 여러 개 사용할 수도 있음
        - 웹 문서에서 이미지나 텍스트 등의 요소를 제어하는 경우가 많음
            - 되도록이면 이미지나 텍스트 등을 다 표시한 후 실행하는 것이 좋음
                - </body> 태그 직전에 자바스크립트 코드를 삽입함
    2) 외부 스크립트 파일로 연결해서 자바스크립트 작성하기
        - <script> 태그 없이 자바스크립트 코드만 작성하고 확장자는 *.js 파일로 저장함
        - <script> 태그의 src 속성을 이용해서 자바스크립트 파일을 연결하면 됨
        - 기본형
            - <script src="외부 스크립트 파일 경로"></script>
