1. 자바스크립트에서 객체
    1) object
    2) 프로그램에서 인식할 수 있는 모든 대상을 가리킴
    3) 현실에 존재하는 대상이라면 어떠한 것도 컴퓨터에서 인식할 수 있는 객체로 바꾸어서 표현 가능
    4) 객체는 데이터를 저장하고 처리하는 기본 단위
    5) 예)
        - 자동차
            - 제조사, 모델명, 색상, 배기량 등
            - 시동 걸기, 달리기, 멈추기 등
        - 회원
            - 이름, 나이, 사는 지역, 가입 날짜 등
            - 물건 구매, 평점 입력 등

2. 내장 객체
    1) built-in object
    2) 프로그래밍할 때 자주 사용하는 요소는 미리 객체로 정의되어 있음
    3) 예)
        - Date 객체
        - Number, Boolean, Array, Math 등

3. DOM (문서 객체 모델 = Document Object Model)
    1) 웹 문서 자체도 객체
    2) 웹 문서에 포함된 이미지와 링크, 텍스트 필드 등은 각각 별도의 객체임

4. 웹 브라우저 객체
    1) 웹 브라우저에서 사용하는 정보도 객체로 지정되어 있음
    2) 예)
        - navigator 객체, history 객체, location 객체, screen 객체 등

5. 사용자 정의 객체
    1) 사용자가 필요할 때마다 자신의 객체를 정의해서 사용
    2) 예)
        let book = {
            title: "자바스크립트",
            page: 500;
        }
