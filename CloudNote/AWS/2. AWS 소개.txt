1. 클라우드 컴퓨팅
    1) 컴퓨팅이란 서버나 스토리지, 데이터베이스 등 컴퓨터 자원을 필요한 만큼만 원격으로 빌려서 사용할 수 있도록 하는 기술
    2) AWS, GCP, Azure

2. AWS
    1) 아마존 웹 서비스 (Amazon Web Services)
    2) 2006년부터 서비스 시작한 클라우드 컴퓨팅 플랫폼
        - 웹 사이트 호스팅으로 시작함
        - DB, storage, AI 등 다양한 서비스 제공함

3. AWS를 알아야 하는 이유
    1) 높은 시장 점유율 : 국내 기업 6~70% 이상 사용중
    2) 다양한 서비스와 통합 옵션
    3) 글로벌 인프라
        - 리전 (Region)
            - 어떤 위치에서도 데이터 처리와 액세스를 빠르고 효율적으로 할 수 있음
            - 지역
            - 전세계에서 데이터 센터를 클러스트링하는 물리적 위치
            - AWS가 컴퓨터들을 설치해 놓은 위치
        - 리전의 핵심 특징
            - 전세계적으로 다양한 리전 보유
            - 리전마다 고유 코드가 배정되어 있음
                - 서울 : ap-northeast-2
        - 리전 선택 기준
            - 서비스를 주로 사용하는 사람들과 가까운 위치로 선택함

4. AWS 서비스 6가지
    1) EC2
        - 원격으로 접속해서 사용할 수 있는 컴퓨터를 빌려주는 서비스
        - Amazon Elastic Compute Cloud
    2) Route 53
        - 도메인 주소를 발급하고 관리하는 서비스
        - 영문이나 한글 등 문자로 이루어진 컴퓨터 주소를 도메인이라고 함
        - 도메인 구매부터 관리까지 도메인과 관련된 모든 기능을 제공함
    3) Elastic Load Balancing (ELB) - 과금서비스
        - 웹 사이트로 보내는 요청을 연결된 서버로 적절히 나누는 서비스
        - HTTPS를 쉽게 설정할 수 있음
    4) RDS - 과금서비스
        - 관계형 데이터베이스를 빌려주는 서비스
        - 백엔드 서버에서 데이터를 안정적으로 저장하고 처리하려면 데이터베이스가 필요함
    5) S3
        - 파일을 안전하고 효율적으로 저장할 수 있는 스토리지를 빌려주는 서비스
        - 파일이나 이미지를 저장하는 기능
    6) CloudFront
        - 파일이나 동영상 등의 컨텐츠를 빠르게 전송하도록 도와주는 서비스
