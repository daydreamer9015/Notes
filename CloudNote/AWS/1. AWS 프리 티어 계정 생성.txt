1. AWS 서비스 사용
    1) 먼저 계정 생성
    2) 계정 생성 시점을 기준으로 12개월 동안 무료 서비스 제공
        - Free Tier
    3) AWS에 가입한 지 12개월 지나면 새로 계정 만들면 다시 무료 사용 가능

2. AWS 계정 만들기
    1) 해외 결제 가능한 카드 필요
    2) 사이트 접속
    3) 계정 정보 입력
    4) 이메일 인증
    5) 연락처 정보 입력
    6) 결제 정보 입력
    7) 문자메시지 인증

3. 로그인하기
    1) 루트 사용자
        - 모든 권한을 다 가진 최고 관리자
    2) IAM 사용자
        - 루트 사용자가 생성할 수 있는 사용자
        - 각 사용자마다 역할을 나눠 필요한 권한을 부여할 수 있음
            - 개발자 : EC2, RDS에 접근할 수 있는 권한을 줌
            - 재무 담당자 : 비용 확인 기능에만 접근할 수 있는 권한을 줌
