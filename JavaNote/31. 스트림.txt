1. 스트림 (Stream)
    1) 자료 처리 기능을 구현한 클래스
    2) 예를 들어 배열 요소를 특정 기준에 따라 정렬(sort)하거나 요소 중 특정 값 제외하고 출력하는(filter) 기능 등
    3) 배열, 컬렉션 등의 자료를 일관성 있게 처리할 수 있음

2. 자바 스트림 API
    1) java.util.stream

3. 스트림 객체 생성하기
    1) 컬렉션에서 .stream() 메서드 호출
    2) Arrays.stream()
    3) Stream.of()

4. 스트림 활용
    1) (정수) 배열
    2) Collection 인터페이스

5. 스트림의 다양한 연산
    1) 자료를 처리하는 데 필요한 여러가지 연산을 메서드로 제공함
    2) 종류
        - 중간 연산
            - 자료를 필터링, 변형하여 또 다른 자료를 내부에서 생성함
        - 최종 연산
            - 생성된 내부 자료를 소모해 가면서 연산을 수행함
            - 마지막에 한번만 호출됨
            - 최종 연산이 호출되어야 중간 연산의 결과가 만들어짐
    3) 중간 연산
        - Stream<T> filter(Predicate<? super T> predicate)
            - 조건에 맞는 요소만 골라내는 메서드
            - 특정 조건이 true인 요소만 골라내 새로운 스트림을 만들어냄
            - Predicate라는 함수형 인터페이스가 필요함
        - <R> Stream<R> map(Function<? super T,? extends R> mapper)
            - map() 연산 수행 시 사용하는 Function<T,R>
                - 제너릭 자료형을 두 개(T, R) 사용
                - 하나는 입력 자료형, 다른 하나는 출력 자료형
                - T 자료형의 인자를 바당 R 자료형으로 출력한다는 의미
        - Stream<T> sorted(Comparator<? super T> comparator)
            - 요소를 정렬하는 연산 메서드
            - 비교할 때는 어떤 기준으로 정렬할 것인지 정해서 구현해야 함
