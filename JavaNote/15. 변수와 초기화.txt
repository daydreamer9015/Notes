1. 변수의 종류
    1) 지역 변수
        - 메서드에 선언, 매개변수도 지역 변수의 한 종류임
    2) 멤버 변수(필드)
        - 클래스에 선언

2. 변수의 값 초기화
    1) 지역 변수
        - 초기화를 하지 않으면 컴파일 에러가 남. !!!반드시 수동으로 초기화 해야함!!!
    2) 멤버 변수
        - 값을 부여하지 않아도 자동 초기화가 됨
        - 기본값: 숫자(int) = 0, boolean = false, 참조형 = null
            - null 값은 참조할 대상이 없다는 의미. 'null'이라는 값이 들어있는 것임
        - 개발자가 초기값을 직접 지정할 수 있음

3. null
    1) 참조형 변수에는 항상 객체가 있는 위치를 가리키는 참조값이 들어감
    2) 아직 가리키는 대상이 없거나 혹은 가리키는 대상을 나중에 입력하고 싶은 경우 = null이 나옴
    3) 참조형 변수에서 아직 가리키는 대상이 없다면 null이라는 값을 넣을 수 있음
        - null은 값이 존재하지 않는, 값이 없다는 뜻

4. NullPointerException
    1) 주소가 없는 곳을 찾아갈 때 발생하는 예외 - 흔히 발생할 수 있는 critical error
    2) 멤버 변수와 null
