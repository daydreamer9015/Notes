1. 조건문
    1) if문, switch문
    2) 특정 조건에 따라서 다른 코드를 실행하는 것

2. if문
    1) if(condition) {
        // 조건이 참일 때 실행되는 코드
        }

3. else문
    1) if문에서 만족하는 조건이 없을 때 실행하는 코드 제공
    2) if(condition) {
        // 조건이 참일 때 실행되는 코드
        } else {
            // 만족하는 조건이 없을 때 실행되는 코드
        }

4. else if문
    1) 앞선 if문의 조건이 거짓일 때 다음 조건을 검사함
    2) 만약 앞선 if문이 참이라면 else if문을 실행하지 않음
    3) if(condition1) {
        // 조건1이 참일 때 실행되는 코드
    } else if(condition2) {
        // 조건1이 거짓이고, 조건2가 참일 때 실행되는 코드
    } else if(condition3) {
        // 조건2가 거짓이고 조건3이 참일 때 실행되는 코드
    } else {
        // 모든 조건이 거짓일 때 실행되는 코드
    }
    4) if문 전체를 하나로 묶음
        - 특정 조건이 만족되면 해당 코드를 실행하고 if문 전체를 빠져나옴
        - 특정 조건을 만족하지 않으면 다음 조건을 검사함
            - 순서대로 맞는 조건을 찾아보고 맞는 조건이 있으면 딱 1개만 실행됨
        - else는 생략 가능함

5. if문과 else if문
    1) 서로 연관된 조건일 때 함께 사용함
    2) 서로 관련이 없는 독립 조건이면 else if를 사용하지 않고 if문을 각각 따로 사용해야 함

6. if문 {}중괄호 생략
    1) if문 다음에 실행할 명령이 하나만 있는 경우엔 생략 가능
    2) else if, else도 마찬가지임
    3) 생략 여부 결정 : 가독성, 유지보수성을 감안
