1. JDK, JRE
    1) JDK
        - Java Development Kit
        - 개발 환경
        - JRE를 포함
        - 컴파일러 등 프로그램 개발 도구가 포함
        - 버전 중요(언어버전)
    2) Java SE 스펙 준수
        - Oracle JDK
        - Open JDK
    3) JRE
        - Java Runtime Environment
        - 실행 환경
        - Java 프로그램을 실행하는데 필요한 패키지
        - JVM(Java Virtual Machine), 각종 명령을 포함
        - 클래스 라이브러리

2. IDE (Intergated Development Environment)
    1) 이클립스(Eclipse)
        - 오픈소스 (by IBM)
    2) 인텔리제이(IntelliJ)
        - 커뮤니티 버전(무료), 얼티메이트 버전(상용)

3. 주석(comment)
    1) 소스코드에 대한 이해를 돕기 위해서 설명을 적어둘 때 사용
    2) 종류
        - // : 한 줄 주석. 기호 이후의 모든 텍스트는 주석 처리됨
        - /* */ : 여러 줄 주석. 이 사이의 모든 텍스트는 주석 처리됨

5. 컴파일과 실행
    1) 자바 프로그램은 컴파일과 실행 단계를 거침
    2) Hello.java와 같은 자바 소스코드를 작성함
    3) 자바 컴파일러를 사용해서 소스코드를 컴파일함
        - 자바가 제공하는 javac라는 프로그램을 사용함
        - .java => .class 파일이 생성됨
        - 자바 소스코드를 바이트코드로 변환하며 자바 가상 머신(JVM)에서 더 빠르게 실행될 수 있게 최적화하고 문법 오류 검출함
    4) 자바 프로그램을 실행함
        - 자바가 제공하는 java라는 프로그램을 사용함
        - 자바 가상 머신이 실행되면서 프로그램이 작동함
