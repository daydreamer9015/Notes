1. 패키지(package)
    1) 자바 파일을 구분하기 위한 폴더로 이해하기
    2) 패키지를 만들었다면, 해당 패키지에 들어가는 자바 파일 첫 줄에 package 패키지명; 과 같이 소속된 패키지명이 선언됨

2. 변수(variable)
    1) 변할 수 있는 값을 저장하기 위한 용도로 사용함
    2) 변수 선언
        - 예) int a;
            - 숫자 정수(integer)를 보관할 수 있는 이름이 a라는 데이터 저장소를 만듦
    3) 변수에 값 대입
        - 예) a = 10;
            - =은 오른쪽에 있는 값을 왼쪽에 저장함
    4) 변수 값 읽기
        - System.out.println(a);    // 변수 a의 값을 읽고 출력
    5) 변수 값 변경
        - 변수는 변할 수 있는 수임
    6) 변수는 반드시 초기화해야함

3. 변수 타입(type)
    1) 변수는 데이터를 다루는 종류에 따라 다양한 형식이 존재함
        - int : 정수를 다룸
        - double : 실수를 다룸
        - boolean : 불리언 타입, true, false 값만 사용 가능함
        - char : 문자 하나를 다루며 작은따옴표를 사용해야 함
        - String : 문자열을 다루며 큰따옴표를 사용해야 함
    2) 리터럴(literal)
        - 고정된 값
        - 정수 리터럴, 실수 리터럴, 불리언 리터럴, 문자 하나 리터럴, 문자열 리터럴

4. 숫자 타입
