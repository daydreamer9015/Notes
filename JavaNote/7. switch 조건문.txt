1. switch문
    1) if문을 조금 더 편리하게 사용할 수 있는 기능
    2) if문은 비교 연산자를 사용할 수 있지만, 
       switch문은 단순히 값이 같은지만 비교 가능
    3) 구문
        switch(조건식){
            case value1 : 
            // 조건식 결과값이 value1 일 때만 실행되는 코드
            break;

            case value2 : 
            // 조건식 결과값이 value2 일 때만 실행되는 코드
            break;
            
            default:
            // 조건식의 결과값이 위의 어떤 값에도 해당하지 않았을 때 
            // 실행되는 코드
        }


        - 조건식의 결과값이 어떤 case의 값과 일치하면 해당 case의 코드가 실행됨
        - break 문은 현재 실행 중인 코드를 끝내고 switch문을 빠져나가게 하는 역할
        - 만약 break문이 없으면, 일치하는 case 이후의 모든 case 코드들이 순서대로 실행됨
        - default는 if문의 else와 같음. default구문은 선택사항

2. if문과 switch문
    1) switch문은 조건식이 특정 case와 같은지만 체크할 수 있음
    2) if문은 참, 거짓의 결과가 나오는 조건식을 자유롭게 작성할 수 있음
