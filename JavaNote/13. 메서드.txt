2. 메서드 (Method)
    1) 자바에서는 함수를 메서드라 함
    2) 크게 메서드 선언과 메서드 본문으로 나눌 수 있음
        - 메서드 선언(Method Declaration)
            - 예) private static double add(double num1, double num2);
                -----------------
                - private, public
                    - 다른 클래스에서 호출할 수 있는 메서드(public)
                    - 다른 클래스에서 호출할 수 없는 메서드(private)
                - static
                    - 객체를 생성하지 않고 호출할 수 있는 정적 메서드라는 의미
                
                - double add(double num1, double num2)
                    - double : 반환 타입(리턴 타입)을 의미함. 메서드의 실행 결과를 반환할 때 사용할 반환 타입
                    - add : 메서드 이름 부여. 이 이름으로 메서드 호출
                    - (double num1, double num2) : 메서드 호출할 때 전달하는 입력값을 정의함. 이 변수들은 해당 메서드 안에서만 사용됨. 영어로 파라미터(parameter), 한글로 매개변수라 함
        - 메서드 본문(Method Body)
            - 메서드가 실제 수행해야 하는 코드블럭 {}
            - 메서드를 호출하면 메서드 본문이 순서대로 실행됨
            - 메서드 본문은 블랙박스임
                - 메서드를 호출하는 곳에서는 메서드 선언은 알지만 메서드 본문(구현 부분)은 모름
            - 메서드의 실행 결과를 반환하려면 return문을 사용하면 됨
                - return문 다음에 반환할 결과를 적어주면 됨
    3) 메서드 호출
        - 정의한 메서드를 호출해서 실행함
        - 메서드 이름에 입력값을 전달함
        - 예) double result = add(num1, num2);
            - add라는 메서드에 숫자 5, 10을 전달하면서 호출함
            => 메서드를 호출하면 메서드는 계산을 끝내고 결과를 반환함
        - 인수(Argument)
            - 인자, 아규먼트라고도 부름
            - 함수를 호출하면서 넘기는 값
            - 메서드 호출할 때 인수를 넘기면 그 인수가 매개변수에 대입됨
        - 매개변수 (Parameter)
            - 메서드 정의할 때 선언한 변수를 매개변수라고 함

3. 메서드 정의
    1) 정의
        - 제어자 반환타입 메서드이름(매개변수 목록) {
            메서드 본문
        }
    2) 구분
        - 매개변수가 없거나 반환 타입이 없는 경우
            - 매개변수를 비워두고 정의함
            - 반환 타입을 void로 정의하면 됨
            - 반환 타입
                - 반환 타입이 있으면 반드시 값을 반환해야 함
4. 메서드 호출과 값 전달
    1) 자바는 항상 변수의 값이 있을 때 이것을 다이렉트로 전달하지 않고 복사해서 대입함
        - 참조형 변수에 있는 값인 참조값(주소값)을 복사해서 대입함
    2) 메서드를 사용해서 값을 변경하려면 메서드의 호출 결과를 반환 받아서 사용하면 됨
