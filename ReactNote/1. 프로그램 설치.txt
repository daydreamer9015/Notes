6. 코드 스니펫(snippet) 명령어 (React 컴포넌트 생성)
    1) rsc
        - React Stateless Component
        - 상태(state)가 없는 함수형 컴포넌트 생성
        - 단순한 UI 컴포넌트 생성할 때 사용
    2) rfce
        - React Functional Component with Export
        - function 키워드를 사용한 함수형 컴포넌트 생성
        - 마지막 줄에 export default를 추가함
        - 상태(state)와 훅(hook)을 사용할 수 있는 일반적인 컴포넌트 생성
